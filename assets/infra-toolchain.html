⭐ À installer en premier (gratuit · fort impact)

  1. Couverture de tests ≥ 50 % : Vitest+RTL (front), JUnit+JaCoCo (back) — exigence RNCP C18/C25.
  2. Sécurité en CI bloquante : Trivy + Semgrep + Gitleaks + OWASP Dependency-Check.
  3. Dependabot / Renovate : maj automatique des dépendances.
  4. Hébergement : Oracle Cloud Always Free + Caddy (TLS auto Let's Encrypt).
  5. Observabilité : SigNoz (déjà là) + Uptime Kuma + sauvegarde restic testée.
